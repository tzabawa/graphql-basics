# Title

graphql-basics

## Description

Graphql-basics is an api showcasing the basic functions of graphql.

## How to install and run project

In the root directory of the project run the following commands:

```bash
npm install
```

```bash
npm run start
```

The api is running on <b>localhost:4000</b>.
